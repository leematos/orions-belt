# Orion's Belt

## What it does

This extension is based on [lean-tab-limiter](https://github.com/eguerlain/lean-tab-limiter) by Etienne Guerlain.

This version is specifically for the Orion browser, with 12 tabs which are the amount that fit on an iPhone tab list without scrolling.
